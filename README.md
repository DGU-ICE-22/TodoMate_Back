# Backend for TODOMATE


### 구현 화면


1. 메인 페이지 
    1. 로그인이 되어있지 않은 상태라면 메인 페이지에서는 로그인 및 회원가입 버튼이 크게 뜨도록 함. ( 나중에 구현 )
    2. 로그인 후 투두 시작하기 버튼 누르면 투두 화면으로 이동
2. 투두 페이지
    1. Todo 앱 기능 정의서를 토대로 디자인
3. 로그인 페이지
4. 회원가입 페이지
5. 프로필 페이지
